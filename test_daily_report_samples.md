## テストパターン1: 詳細な商談（理想的）
**訪問先**: ○○農園
**製品**: パワーガイザー液剤

**商談内容**:
```
本日、○○農園の代表である田中太郎様を訪問しました。田中様は大豆を5haで栽培されており、毎年雑草防除に苦労されているとのことでした。現状は手作業で除草しており、1haあたり約2日間の労力がかかっているそうです。

パワーガイザー液剤を紹介したところ、豆類専用の除草剤で作物への影響が少ない点に強い関心を示されました。特に、畦間処理が可能な点と、展着剤が不要で作業が簡便な点を評価いただきました。

使用方法として、出芽揃期（6月中旬予定）に300ml/10aを散布する計画を提案しました。田中様からは「まずは1haで試験してみたい」とのご意向をいただきました。

ただし、圃場の隣に水稲があるため、ドリフト対策としてドリフト軽減ノズルの使用が必須であることを説明しました。また、散布適期の見極めが重要で、雑草が大きくなりすぎると効果が低下することもお伝えしました。

次のステップとして、来週6月10日に再訪問し、1ha圃場での試験区を設定します。散布適期には立ち会い、散布方法の指導を行う予定です。効果確認は散布後2~3週間で実施し、満足いただけた場合は残り4haでの導入を目指します。
```

**期待される出力**:
- タイトル: ○○農園での大豆除草剤提案
- 訪問先: ○○農園
- 参加者: 代表 田中太郎様、営業担当（自分）
- 製品: パワーガイザー®液剤
- 目的: 大豆栽培における雑草防除の課題解決と除草剤導入提案
- 結果: 5ha栽培、手作業除草に2日/haかかっている。1haでの試験導入を検討
- 提案: 出芽揃期（6月中旬）に300ml/10a散布。ドリフト軽減ノズル使用を推奨
- 課題: 隣接水稲へのドリフト対策が必須。散布適期の見極めが重要
- 次のステップ: 6月10日再訪問、試験区設定。散布時に立ち会い指導。効果確認後、残り4haへの展開を目指す

## テストパターン6: 専門用語多用
**訪問先**: ◎◎農場
**製品**: パワーガイザー液剤

**商談内容**:
```
◎◎農場の大豆圃場を訪問。現在出芽揃期で、一年生広葉雑草が発生始期～2葉期の状態でした。パワーガイザー液剤の散布適期であることを説明し、300ml/10aでの処理を提案しました。

圃場は粘土質で排水性がやや悪いため、土壌が適湿な状態で散布することを推奨しました。また、有機リン系殺虫剤を先週散布したばかりとのことで、10日以上空けてから散布する必要があることをお伝えしました。

顧客からは「初生葉期以降の散布だと薬害が出ないか」との質問がありましたが、一時的な黄化は発生する可能性があるものの、その後の生育には影響しないことを説明しました。

2週間後に効果確認のため再訪問する予定です。
```

**期待される出力**:
- 農業用語（出芽揃期、発生始期、一年生広葉雑草、粘土質、有機リン系等）を正確に理解
- 技術的な詳細を適切に記載
- 注意事項（土壌適湿、10日間隔、薬害）を明確に

## 評価基準

1. **正確性**: 製品名（®マーク含む）、数値、日付が正確か
2. **詳細性**: 各項目が2~5文程度で詳しく記載されているか
3. **具体性**: 抽象的でなく具体的な記述になっているか
4. **論理性**: 目的→結果→提案→課題→次のステップの流れが自然か
5. **実用性**: 後で見返したときにアクションが分かるか
6. **補完性**: 情報不足の場合に合理的に補完されているか
7. **専門用語**: 農業用語が正確に使われているか
